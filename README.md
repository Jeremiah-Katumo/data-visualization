# Data Visualization with Python

This repository contains examples and tutorials for data visualization using Python's popular visualization libraries: Matplotlib and Seaborn.

## Project Structure

```
├── matplotlib/
│   ├── 1-quick-start.ipynb
│   ├── 2-figures-and-backends.ipynb
│   ├── 3-backends.ipynb
│   ├── 4-application-interfaces.ipynb
│   └── matplotlibrc
├── seaborn/
│   └── __init__.py
└── requirements.txt
```

## Topics Covered

### Matplotlib Tutorials
1. Quick Start Guide

### Seaborn Integration
- Integration with Matplotlib
- Statistical Data Visualization

## Setup

1. Clone this repository:
```bash
git clone https://github.com/Jeremiah-Katumo/data-visualization.git
cd data-visualization
```

2. Install required packages:
```bash
pip install -r requirements.txt
```

## Usage

The tutorials are provided as Jupyter notebooks in the `matplotlib` directory. To run them:

1. Start Jupyter Notebook:
```bash
jupyter notebook
```

2. Navigate to the `matplotlib` directory and open any of the numbered notebooks to begin learning.
